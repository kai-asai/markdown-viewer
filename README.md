# markdown viewer

Markdown ソースとプレビューを分けずに編集する、一画面ライブレンダリング Markdown エディタです。外部ライブラリなしの単体 HTML ですが、ローカル `.md` ファイルの読み込み、保存、書き出しに対応しています。

## 使い方

- `index.html` をダウンロードしてブラウザで開く
- `開く` から `.md` / `.markdown` / `.txt` を読み込む
- 本文ブロックをクリックすると、そのブロックだけ Markdown ソースとして編集できる
- 編集中も同じブロック内にレンダリング結果が即時表示される
- ブロック外をクリック、または Esc でレンダリング表示に戻る
- アウトラインはデスクトップ幅ではスクロールに追従する
- `全文ソース` は全画面で開き、長い Markdown をまとめて編集できる
- 見出し、箇条書き、チェックリスト、引用、表、コードブロック、リンク、画像、取り消し線をレンダリングする
- 内容とタイトルは `localStorage` に自動保存される
- Chrome / Edge など File System Access API 対応ブラウザでは、読み込んだファイルに `保存` で上書きできる
- 非対応ブラウザや直接上書きできない場合は、`書き出し` で `.md` としてダウンロードする
- `PDF` で印刷用プレビューを開き、ブラウザの印刷画面から `PDFに保存` できる
- `コピー` で Markdown 本文をクリップボードへコピーする
- 下書きはブラウザの `localStorage` に保存されるため、共有PCでは注意が必要
- 生HTMLは安全のためエスケープされる設計

外部ライブラリ、CDN、ビルド手順はありません。
